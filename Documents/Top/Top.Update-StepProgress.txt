Update-StepProgress Function Documentation

Description:
Updates the bottom-level progress bar and subtask label text to reflect progress inside the current pipeline step.

Input Type:
1. PercentComplete [int] — Progress percentage (0 to 100).
2. SubTaskMessage [string] — Descriptive message for the current subtask.

Output Type:
✅ [void] — Updates the GUI step progress bar and subtask label.

Algorithm (Pseudo-code):
1. Check if the progress form exists and is not disposed.
2. Set the step progress bar value, clamped to 100 max.
3. Update the subtask label text with the given message.
4. Process UI events to refresh the display.

Function Calls:
None

Possible Issues:
✅ Covered: Avoids updates if form is not present.
❌ Not Covered: No thread-safety guarantees.
