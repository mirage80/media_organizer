Close-TwoLevelProgressBar Function Documentation

Description:
Closes and disposes of the two-level progress bar window, cleaning up all global references.

Input Type:
None

Output Type:
✅ [void] — Closes the progress window and cleans up resources.

Algorithm (Pseudo-code):
1. Check if the progress form exists and is not disposed.
2. Call Close() and Dispose() on the form.
3. Nullify all global progress bar and label variables.

Function Calls:
None

Possible Issues:
✅ Covered: Handles disposal gracefully.
❌ Not Covered: No checks for window visibility before closing.
