Function Name:
MediaTools.Make_zero_GeoTag

Description:
Creates a default GPS metadata object with zeroed values, used when valid GPS data is unavailable.

Input Fields:
None

Output Type:
✅ [hashtable] — A GPS metadata object with zeroed latitude and longitude
❌ Not applicable (no exception expected)

Algorithm:
1. Create an empty hashtable
2. Add the following key-value pairs:
   2.1 🔹 GPSLatitudeRef: "N"
   2.2 🔹 GPSLatitude: 0
   2.3 🔹 GPSLongitudeRef: "E"
   2.4 🔹 GPSLongitude: 0
3. ✅ Return the hashtable

Possible Issues:
* ✅ Default direction is always "N" and "E", not configurable
* ✅ Zero coordinates may still appear valid to naive systems
* ❌ No validation against real-world GPS bounds or nulls
