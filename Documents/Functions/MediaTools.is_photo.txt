Function Name:
MediaTools.is_photo

Description:
Determines whether a given file extension corresponds to a known photo/image format.

Input Fields:
1. extension [string] — File extension (e.g., ".jpg", ".png")

Output Type:
✅ [bool] — True if the extension is a known photo format
❌ False otherwise

Algorithm:
1. Normalize input extension
  1.1 Convert to lowercase
2. Compare against known photo extensions list:
  a. ".jpg"
  b. ".jpeg"
  c. ".png"
  d. ".heic"
  e. ".webp"
  f. ".tiff"
  g. ".bmp"
3. Return True if extension is found in list
4. Otherwise, return False

Possible Issues:
* ✅ Case insensitive matching
* ❌ No wildcard or regex matching
* ❌ Relies on static list of extensions
