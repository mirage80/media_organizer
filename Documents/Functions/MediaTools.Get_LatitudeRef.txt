Function:
MediaTools.Get_LatitudeRef

Description:
Extracts latitude reference ("N" or "S") from EXIF GPS data of an image.

Input Type:
1. exif_json [hashtable] — Parsed EXIF JSON data from ExifTool

Output Type:
✅ [string] — "N" or "S"
❌ If missing or malformed, returns empty string

Algorithm:
1. Check if "GPSLatitudeRef" exists in exif_json
  1.1 If yes, return its string value
  1.2 If value is not "N" or "S", return empty string
2. If missing, return empty string

Possible Issues:
✅ Covered: Field missing from EXIF
✅ Covered: Lowercase variants like "n" or "s"
❌ Not Covered: Malformed or non-string values
