Function Name:
MediaTools.is_video

Description:
Determines whether a given file extension corresponds to a known video format.

Input Fields:
1. extension [string] — File extension (e.g., ".mp4", ".mov")

Output Type:
✅ [bool] — True if the extension is a known video format
❌ False otherwise

Algorithm:
1. Normalize input extension
  1.1 Convert to lowercase
2. Compare against known video extensions list:
  a. ".mp4"
  b. ".mov"
  c. ".avi"
  d. ".mkv"
  e. ".wmv"
  f. ".flv"
  g. ".webm"
3. Return True if extension is found in list
4. Otherwise, return False

Possible Issues:
* ✅ Case insensitive matching
* ❌ No wildcard or regex matching
* ❌ Relies on static list of extensions
