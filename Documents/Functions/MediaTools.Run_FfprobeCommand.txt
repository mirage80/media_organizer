Function Name:
MediaTools.Run_FfprobeCommand

Input:
1. filePath [string] — Path to the video file
2. fields [string[]] — List of metadata fields to extract
3. timeoutSec [int] — Maximum allowed time for the command to run

Output:
✅ [hashtable] — Parsed ffprobe metadata
❌ Returns $null on error

Algorithm:
1. If filePath does not exist
   1.1 Return $null
2. Construct ffprobe JSON command with selected fields
3. Use Start-Process to run ffprobe asynchronously
   3.1 Set a timeout
   3.2 Capture stdout to temp file
4. If timeout occurs
   4.1 Kill ffprobe
   4.2 Return $null
5. Parse JSON output from temp file
   5.1 If parsing fails
       5.1.1 Return $null
6. Extract requested fields
7. Return hashtable

Possible Issues:
✅ Covered: Nonexistent file path
✅ Covered: Timeout behavior
✅ Covered: JSON parsing errors
❌ Not Covered: Partial metadata extraction failures
❌ Not Covered: Conflicts in field name casing
