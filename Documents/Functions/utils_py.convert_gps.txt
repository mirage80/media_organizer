Function Name:
utils.convert_gps

Description:
Converts GPS coordinate tuple from degrees, minutes, seconds (DMS) format into decimal degrees.

Input Fields:
1. value [tuple of 3 floats] — A 3-tuple representing degrees, minutes, and seconds
2. ref [string] — Reference direction ('N', 'S', 'E', or 'W')

Output Type:
✅ [float] — Decimal representation of coordinate
❌ Returns 0.0 for invalid input or unsupported ref

Algorithm:
1. Validate input
  1.1 Ensure value is a 3-tuple
  1.2 Ensure ref is one of 'N', 'S', 'E', 'W'
2. Convert to decimal
  2.1 decimal = degrees + (minutes / 60.0) + (seconds / 3600.0)
3. Adjust sign based on ref
  3.1 If ref in ['S', 'W'], multiply decimal by -1
4. Return decimal

Possible Issues:
* ✅ Handles negative coordinates using ref
* ❌ Does not handle malformed tuples
* ❌ Returns 0.0 silently on error instead of raising
* ❌ No support for fractional seconds as rational numbers
